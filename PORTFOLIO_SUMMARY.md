# Portfolio Summary — Samandeep Kaur | Junior Data Analyst
Thames Valley, UK | Python | SQL | Power BI | MBA Business Analytics

## Project 1: Retail Deep Dive

This project analysed a UK e-commerce retail dataset of 400,947 transactions generating 8.8M in revenue across 4,314 customers. Using Python for data cleaning and feature engineering, SQL for behavioural segmentation, and Power BI for executive dashboards, the analysis identified that 67.06% of customers were repeat purchasers and segmented the customer base into six RFM tiers, including 460 Champions driving a disproportionate share of revenue. The resulting Power BI dashboard enables a retail leadership team to monitor revenue trends, prioritise high-value customer retention, and act on segment-level insights without touching the underlying data.

## Project 2: AI Marketing Pipeline

Businesses receiving thousands of customer reviews lack the capacity to analyse sentiment manually at scale. This project built an end-to-end AI-powered sentiment analysis pipeline using Python and a pre-trained NLP model to classify customer reviews as positive, neutral, or negative automatically. The pipeline processed review data in batch, aggregated sentiment scores by product category and time period, and produced a structured automated report requiring zero manual intervention. A marketing team can now receive a weekly sentiment briefing in minutes rather than days, enabling faster response to emerging customer dissatisfaction before it impacts retention or brand reputation.

## Project 3: ROI Optimizer — Marketing Mix Model

Marketing teams routinely allocate budgets without quantitative evidence of which channels deliver the strongest return. This project built a Marketing Mix Model in Python that applies adstock transformation to account for the delayed and decaying effect of advertising spend across TV, social media, paid search, and email channels. The model produced a channel-level ROI breakdown and a budget allocation recommendation, identifying which channel generates the highest revenue return per 1,000 spent. A scenario analysis component allows a marketing director to simulate the revenue impact of reallocating budget between channels before committing spend, turning intuition-based decisions into data-driven ones.
